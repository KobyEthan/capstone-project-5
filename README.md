## Run Locally

Clone the project

```bash
  git clone https://github.com/KobyEthan/persistence-list.git persistence-list
```

Go to the project directory

```bash
  cd persistence-list
```

Install dependencies

```bash
  npm install
```
Run project

```bash
  npm start
```

Go to localhost:3000 and view the project.
